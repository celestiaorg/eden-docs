# Use the docs with AI

Eden publishes a public, read-only Model Context Protocol server for these docs.
Add it to your coding agent to search the docs, read pages as Markdown, and keep
Eden network and tooling details close at hand while you build.

| Property  | Value                                    |
| --------- | ---------------------------------------- |
| Name      | `eden-docs`                              |
| URL       | `https://docs.eden.zone/mcp`             |
| Transport | Streamable HTTP                          |
| Tools     | `list_pages`, `read_page`, `search_docs` |

:::info
The docs MCP server is public and read-only. It never needs a wallet recovery
phrase, private key, API key, or `.env` file.
:::

## Codex

```bash
codex mcp add eden-docs --url https://docs.eden.zone/mcp
```

Codex stores MCP servers in `~/.codex/config.toml`. To configure the server by
hand, add:

```toml
[mcp_servers."eden-docs"]
url = "https://docs.eden.zone/mcp"
```

Inside Codex, run `/mcp` to confirm the server is connected.

## Claude Code

```bash
claude mcp add --transport http eden-docs https://docs.eden.zone/mcp
```

Inside Claude Code, run `/mcp` to confirm the server is connected.

## Cursor

Add this to `.cursor/mcp.json` for this project, or to `~/.cursor/mcp.json` to
use it everywhere:

```json
{
  "mcpServers": {
    "eden-docs": {
      "url": "https://docs.eden.zone/mcp"
    }
  }
}
```

## VS Code / GitHub Copilot

Add this to `.vscode/mcp.json` for this workspace, or to your user MCP config:

```json
{
  "servers": {
    "eden-docs": {
      "type": "http",
      "url": "https://docs.eden.zone/mcp"
    }
  }
}
```

## Windsurf / Cascade

Add this to `~/.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "eden-docs": {
      "serverUrl": "https://docs.eden.zone/mcp"
    }
  }
}
```

## Try it

After connecting the server, ask your agent:

```text
Search the Eden docs for the mainnet RPC URL and block explorer.
```
