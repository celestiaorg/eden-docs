# Eden mainnet

## Chain info

| Name            | Value                                                      |
| --------------- | ---------------------------------------------------------- |
| Chain ID        | `714`                                                      |
| Block time      | `100ms`                                                    |
| Block gas limit | `50,000,000`                                               |
| Native token    | TIA (18 decimals)                                          |
| Chainlist       | [chainlist.org/chain/714](https://chainlist.org/chain/714) |

You can add Eden to your wallet directly from [Chainlist](https://chainlist.org/chain/714), or see
the [wallets guide](/tooling/wallets) for manual setup.

:::info
The public RPC confirmed chain ID `0x2ca` (`714`) and latest-block gas limit `0x2faf080`
(`50,000,000`) during the docs update pass.
:::

## RPCs

| Endpoint                       | Type          | Provider                         |
| ------------------------------ | ------------- | -------------------------------- |
| `https://rpc.eden.gateway.fm/` | HTTP RPC      | [Gateway.fm](https://gateway.fm) |
| `wss://rpc.eden.gateway.fm/ws` | WebSocket RPC | [Gateway.fm](https://gateway.fm) |

Gateway.fm public RPCs have a rate limit of 30 requests per second per IP address. Higher limits
are available through [admin.gateway.fm](https://admin.gateway.fm).
