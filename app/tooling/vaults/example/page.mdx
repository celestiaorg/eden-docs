# Example with token-vaults

This guide demonstrates deploying an ERC-4626 vault on Eden using the [token-vaults](https://github.com/gbarros/token-vaults) repository, which includes a complete vault implementation with deployment scripts and a frontend demo.

## What you'll deploy

- **ERC-4626 vault**: Synchronous deposit/redeem interface
- **Oracle integration**: Price feeds for vault assets
- **Market infrastructure**: Lending market for the vault strategy
- **Frontend demo**: UI for interacting with the vault

## Quick start

Follow the [quick start guide](https://github.com/gbarros/token-vaults?tab=readme-ov-file#-quick-start) in the repository to:

1. Clone the repository and set up your environment
2. Deploy the contracts using Forge scripts
3. Run the frontend demo locally
4. Interact with your vault through the UI

The repository includes detailed instructions for each step, including prerequisites, environment setup, and deployment scripts.

## Customization

The repository provides a working example that you can customize:

- **Vault strategy**: Modify the contracts in `contracts/src/` to implement your own yield strategy
- **Frontend**: Customize the UI in `frontend/` or integrate the vault into your own application
- **Additional features**: Add time locks, withdrawal fees, or strategy-specific logic

For implementation details and troubleshooting, see the [repository documentation](https://github.com/gbarros/token-vaults).
