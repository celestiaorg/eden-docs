# Storage

Data storage, not to be confused with data availability, is an important part of smart contracts. While data availability ensures that information is published and retrievable, storage is about keeping data accessible over time.

There is no official storage provider for Eden. Instead, developers can use third-party solutions. Some common options include:

- [IPFS](https://ipfs.io/) — a peer-to-peer network for content-addressed storage
- [Arweave](https://arweave.org/) — permanent, immutable storage
- [Pinata.cloud](https://pinata.cloud) — an IPFS pinning and management service

Alternatively, developers can also host their assets on Celestia with [Twinkle](https://t.tech) - infrastructure for posting and retrieving blobs on Celestia's data availability layer with a REST API.
