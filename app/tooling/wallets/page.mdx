# Wallets

Eden is an EVM-compatible chains. You can use MetaMask, WalletConnect, Rainbow, or any other EVM wallet to interact with Eden.

## Adding Eden to MetaMask

import { Steps } from 'nextra/components'

<Steps>
## Open MetaMask

Launch MetaMask in your browser and ensure you're logged in.

## Add network

Click the network dropdown (usually shows "Ethereum Mainnet") and select "Add network".

## Manual network addition

Click **"Add a network manually"** in MetaMask and enter these details for the Eden Testnet:

- **Network Name:** Eden Testnet
- **New RPC URL:** https://rpc-evnode-edennet-1-testnet.binary.builders:8443
- **Chain ID:** 1234
- **Currency Symbol:** TIA
- **Block Explorer URL:** https://explorer-edennet-1-testnet.binary.builders:8443/

Or these for mainnet:

- **Network Name:** Eden Mainnet
- **New RPC URL:** TODO
- **Chain ID:** TODO
- **Currency Symbol:** TIA
- **Block Explorer URL:** TODO

<Callout type="info">
  Use an RPC endpoint from the RPCs list ([testnet](/networks/testnet#rpcs) or [mainnet](/networks/mainnet#rpcs)). Some endpoints may be temporarily unavailable, if one fails, try another from the list.
</Callout>

## Save and switch

Click "Save" and MetaMask will automatically switch to Eden testnet.

## Get testnet TIA

If you're using the testnet, visit the [Eden faucet](/networks/testnet#faucet) to get testnet TIA for transactions.
</Steps>

## Other wallets

Most EVM wallets support custom network addition. The process is similar:

1. Find the "Add Network" or "Custom RPC" option
2. Enter the Eden network details above
3. Save and connect

## Troubleshooting

### Network already exists

If you see "Network already exists", the network is already configured in MetaMask.

### RPC endpoint issues

If you get connection errors, try different RPC endpoints from the RPC list ([testnet](/networks/testnet#rpcs) or [mainnet](/networks/mainnet#rpcs)).

### Wrong chain ID

Ensure you're using the correct chain ID: 1234 for testnet, 42420 for mainnet.

That's it! You're ready to use Eden with your wallet.
